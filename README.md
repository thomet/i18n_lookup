It provides a way to see the i18n calls.

# Usage

Put this in your Gemflie

```ruby
gem 'i18n_lookup', :git => 'https://github.com/thomet/i18n_lookup.git'
```
and set the I18N_DEBUG=true e.g.
```
I18N_DEBUG=true rails s
```
